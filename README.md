# Slotify

A web application for booking appointments with professionals.
Built using React and json-server.

# Features
- User registration and login with validation
- Browse services and professionals
- Book appointments
- View and manage your appointments

# Tech Stack
- React 19
- Vite
- React Router DOM
- json-server (mock REST API)

# How to Run

You need two terminals:

**Terminal 1 - React app:**
npm run dev



**Terminal 2 - Database:**
npm run server



Then open http://localhost:5173
